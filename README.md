# Portfolio Website

Simple static portfolio website built with HTML, CSS, and JavaScript.

## Structure

- `index.html` – main page
- `css/styles.css` – stylesheet
- `js/script.js` – JavaScript
- `assets/` – images and other media

## Getting Started

1. Open `index.html` in a browser.
2. Optionally serve with a local web server (`npx serve`, `python -m http.server`, etc.)

Customize the content with your own bio, projects, and contact information.
